# 食物地圖 - 不要玩食物

嘉瑪科技同事中午食物選擇障礙選擇器

## 📱 應用功能

- 🍕 **食物輪盤** - 隨機選擇食物
- 👥 **用戶登入** - 同事帳號管理
- 📊 **管理面板** - 食物數據管理
- 🎨 **現代化界面** - 美觀的用戶體驗

